Sobre os anos iniciais do jogo

O jogo original foi lançado em 1983 e novos jogos da série ainda continuam a ser lançados. Hoje, Bomberman estrela em mais de 60 jogos diferentes incluindo o Bomber Friends.

Bomber Friends é um divertido jogo multiplayer para Android e iOS. em Bomberman, ele tem uma jogabilidade bem parecida com o clássico dos consoles, porém é voltado para se disputar partidas com 2 ou 4 jogadores. O objetivo é simples: bombardear os inimigos, coletar bônus e ganhar recompensas.

Bomberman é um dos maiores clássicos da história dos games e agora Bomber Friends se inspira no jogo para também fazer bastante sucesso, só que nos smartphones. O game utiliza o esquema de cenários e a jogabilidade do clássico para consoles para fazer uma versão multiplayer mobile.

Visualmente, Bomber Friends não impressiona, porém o acabamento é bem parecido com o de Bomberman. Os menus são bem posicionados e a organização dos controles é leve. A usabilidade de Bomber Friends também é ótima, especialmente se o jogador é familiarizado com o Bomberman. > O jogo é simples e com comandos muito fáceis. O único detalhe é que o game não tem tanta história e acaba sendo voltado exclusivamente para batalhas online.
